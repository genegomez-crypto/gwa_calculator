# GWA Calculator

A standalone Python application developed using Tkinter that allows students to calculate their General Weighted Average (GWA).

## Features

- Add multiple subjects and grades
- Calculate GWA automatically
- Determine academic standing
- Simple and user-friendly interface

## Technologies Used

- Python
- Tkinter
